This post discusses deployment strategies for projects created through vibe coding (using LLMs to generate code from plain language descriptions). The article covers three deployment approaches: hosting websites via Vercel, GitHub Pages, or Cloudflare; distributing indie games through Itch.io; and building mobile apps as Progressive Web Apps or native applications, while recommending Twilio Verify for authentication.
